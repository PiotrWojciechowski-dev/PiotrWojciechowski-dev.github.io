About me

I'm Piotr Wojciechowski, a 21 year-old Software engineer. Born on the 8th March 2000 in Poland, currently residing in Ireland. In my 4th Year of college doing a Bsc(Hons) in Computer Science with Software Development.

I have done work in full-stack development, software development, cloud development, data analysis, mobile app creation, database/server management, network management. I like to resolve programming/design issues to create modern, stylish applications that meet user requirements and my personal standards. I also love to create content in Photoshop and Premier pro.

Experienced in languages such as Python, JavaScript, Java, C++ and C#.

Projects

As my final year project, I decided to create an android application that helps users who are faced with words/sentences in a foreign language and don't understand what they mean. The goal of the app is to assist the user in a fast and reliable way by using OCR technology to translate any image given.

During 2nd Year for my final project, I was tasked with two classmates to build a website using the web framework Django. We decided to build a social media web app with an e-commerce section included to showcase what we learned over the year.

An e-commerce website built using Django during my first semester of 2nd Year. It was a group project where we had to show how RESTful methods (GET,POST,PUT and DELETE) work and how a succesful customer lifecycle from sign up to payment operates.

Web application built using Node.js where a user can find information about anime. The project showcases a client and a server where the server would fetch the required data from the sql server deployed in Azure anytime a client requested an api call.

During my on-campus internship I had to create an application to help in analysing palliative care data so predictions could be done for future years. The web app utilises Django and Chart.js to allow medical personal to quickly generate graphs from csv files and have the ability to share them with their colleagues.

EDA project where I analyse alcohol consumption among portugeese students and see its effects on their grades in their mathematics class. Done using Jupyter Notebook.
