About me

I'm Piotr Wojciechowski, a 21 year-old Software engineer. Born on the 8th March 2000 in Poland, currently residing in Ireland. In my 4th Year of college doing a Bsc(Hons) in Computer Science with Software Development.

I have done work in full-stack development, software development, data analysis, mobile app creation, database/server management, network management. I like to resolve programming/design issues to create modern, stylish applications that meet user requirements and my personal standards.
